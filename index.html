Amount of cells

Seeds of cellular automaton
Input cell ids from 0 to (1 - amount of cells) seperated by commas.

Ruleset of cellular automaton
Input in wolfram syntax. For example a serpinski triangle ruleset (rule 90) is the default. Other forms can be found here.

Frames displayed per second

Length/width of each cell in pixels

Notes to play in each octave
Input numerical values of keys in an octave (0-12) which the automaton will play on, seperated by commas. Any key not included will be removed from the keyboard so that a specific scale can be played on. For example, a Minor scale is the default, a blues scale would be: 0, 3, 5, 6, 7, 10
